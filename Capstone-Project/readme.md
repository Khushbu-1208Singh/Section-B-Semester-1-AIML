## Campus Energy-Use Dashboard
A simple Python project that reads electricity usage data of different campus buildings, analyzes it, and creates a visual dashboard showing energy trends.

# Features
• Automatic data loading from multiple CSV files
• Daily & weekly consumption analysis using pandas
• Building-wise summary (mean, min, max, total)
• OOP implementation using Building & MeterReading classes
• Dashboard with 3 charts:
    • Daily trend
    • Weekly comparison
    • Peak-hour scatter plot
• Exports cleaned data, summary table, and a summary text file

# Output Files Generated
• Cleaned_energy_data.csv
• Building_summary.csv
• Summary.txt
• Dashboard.png
