# Daily Calorie Tracker – Python CLI

This mini project is a simple Command Line Calorie Tracker built using Python. The user can enter their meals, log calories, view total and average intake, and check if they are within their daily calorie limit. The tool also gives a warning when the limit is crossed.

# Features

• Add meal names and their calorie values

• Stores data using Python lists

• Calculates total and average calories

• Compares intake with a user-defined daily limit

• Clean and formatted summary output

• (Bonus) Saves the full session report to a .txt file

# What I Learned

• Basic input handling

• Type conversion and arithmetic operations

• Lists and loops

• Conditional statements

• File I/O for saving logs

• Formatting output with f-strings and escape characters

Project Goal - 

To practice core Python concepts by building a small real-world tool that helps track daily calorie intake.
